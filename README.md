## 프로젝트 설명

Sara 프로젝트는 사용자가 구매 결정을 내리는 데 도움을 주는 백엔드 서비스입니다. Django와 Django REST Framework를 기반으로 구축되어 있으며, 간단하고 직관적인 사용자 경험을 제공합니다.
